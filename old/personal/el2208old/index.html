Course Description

From this course you will learn about:
  • Compile, link, and run C codes.
  • Solving mathematical and engineering problems with C.
  • Read and write files in C.
  • Plotting data in MATLAB from C output.

This course is co-requisite of EL2008 - Problem Solving with C. You should enroll in both courses unless you have discussed with the instructor.

General Rules

  • There is no tolerance for being late. We will not allow you to participate if you are coming late.
  • You must wear appropriate clothes: formal shirt, trousers/skirt, and shoes.
  • You should try the lab modules at least once before your lab sessions.
  • You are not required to write any lab report after lab sessions. You must finish the assignment and the assistant will grade your works immediately.
  • Missing any more than 3 lab sessions is grounds for a failing grade.
